The OCFA Graphical User Interface

Web based GUI:
The tactical investigator can view the results without the requirement of installing dedicated software on his workstation.

The graphical user interface of ocfa is based on a collection of cgi scripts and programs running on an Apache webserver.Using this interface it is possible to browse the results on a workstation without installing dedicated ocfa software. Everything is server based, allowing web access to the results.

There are several interfaces to browse through the results. There are interfaces for:

  • Viewing the metadata tree based on the case hierarchy, with the evidence as starting point.
  • Searching for (combinations of) words in the text.
  • A view on the queues/buffers of the anycast router.
  • An overview of the metadata found, usable to search matching evidences.
  • An overview of all graphical images using thumbnails.
  • A collection of time line calendar view.
All views and overviews have the possibility to drill down to the original data and showing the corresponding metadata.

Some of the views are instantly accessibly, also during the operation of the running architecture. Instant accessible views are:

  • Viewing the metadata tree.
  • A view of the anycast buffers.
Other views are only accessible after finishing the process and calling the appropriate commands for creating overviews and optimizing the database for querying. See the quick start reference for information about this.
  • Searching for (combination of) words in the text.
  • An overview of all metadata based on summaries and categories.
  • An overview of all image based data based on thumbnails.
  • A time line view based on a calendar view.
